# 🔩 Hardware: Autonomous Systems and Payload

This directory contains the hardware specifications, CAD models, and electronic circuit designs for the **Aero-Climate Engineering** project.

## Components

### 1. Seeding Capsules
- **Micro-valve control**: Precise release mechanism for silver iodide (AgI) and liquid propane.
- **Environmental shielding**: Resistant to extreme cold and high altitude turbulence.

### 2. UAV Airframe
- **Aerodynamics**: High-lift, high-stability design for stable flight in convective cloud systems.
- **Materials**: Lightweight composite materials for maximum payload capacity.
