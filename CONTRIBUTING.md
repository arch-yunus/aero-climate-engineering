# Contributing to Aero-Climate Engineering

Thank you for your interest in contributing to the **Aero-Climate Engineering** project! We are building the future of autonomous weather modification and edge-AI driven atmospheric control.

## How to Contribute

We welcome contributions of all kinds, including:
- Technical research and documentation
- Edge AI model optimizations
- Hardware design and specifications
- Telemetry and IoT protocol implementations
- Bug fixes and feature requests

### Guidelines

1. **Safety First**: As this project deals with geoengineering and atmospheric control, all proposed changes must adhere to ethical guidelines and safety protocols.
2. **Standardized Process**:
    - Fork the repository.
    - Create a new branch for your feature or fix.
    - Submit a Pull Request with a clear description of your changes.
3. **Code Quality**: Ensure your code follows established patterns and includes necessary documentation.

For detailed development setup, please refer to the specific module's README (e.g., `edge-ai/README.md`).
